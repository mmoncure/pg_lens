# Sample usage
```pglens -i sampleComplex.sql -o temp.json --rw db```

# Install
- sudo npm i -g

# Usage
```
npm run build
pglens:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -i                                                         [string] [required]
  -o                                                                    [string]
      --rw                         [string] [required] [choices: "db", "stdout"]
  -d                                                  [boolean] [default: false]
```
